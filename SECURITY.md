# Security Policy

## Reporting a vulnerability

Please open a **private security advisory** via GitHub's *Report a
vulnerability* (Security tab) rather than a public issue.

## Scope notes

Room & Board is deliberately unauthenticated signage: boards hold widget
preferences and an optional first-name greeting — no accounts, no PII, no
secrets on the client. The security model, hardening measures (output
encoding, CSP, SSRF-safe worker proxying, secret handling), and review
history are described in the README's Security section.
